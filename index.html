PWAインストール方法

「Web / PWA」対応のゲームは、ブラウザからホーム画面に追加するだけで
アプリのように遊べます。以下の手順でインストールしてください。

iPhone / iPad

  1. Safari でゲームのページを開く
  2. 画面下の 共有ボタン（□に↑）をタップ
  3. 「ホーム画面に追加」をタップ
  4. 右上の「追加」をタップして完了！

※ Safari以外のブラウザでは「ホーム画面に追加」が表示されません。

Android

  1. Chrome でゲームのページを開く
  2. 右上の メニュー（︙）をタップ
  3. 「ホーム画面に追加」または「アプリをインストール」をタップ
  4. 確認画面で「インストール」をタップして完了！

※ 「アプリをインストール」と表示される場合もあります。

PC（Chrome / Edge）

  1. Chrome または Edge でゲームのページを開く
  2. アドレスバー右端の インストールアイコン（⊕）をクリック
  3. 「インストール」をクリックして完了！

※ アイコンが表示されない場合はメニュー（︙）→「アプリをインストール」から追加できます。

うまくいかない場合はよくある質問をご確認ください。
